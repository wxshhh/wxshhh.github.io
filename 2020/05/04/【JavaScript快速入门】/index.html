4.2 变量作用域

  • var 定义的变量，在函数体中声明，则在函数体外不可用（不过可以通过闭包在函数体外进行使用）

  • 内部函数可以访问外部函数的成员，反之则不行

  • JavaScript 所有变量声明提到最前面（习惯）

  • 默认所有的全局变量，都自动绑定在 window 对象下

  • 把自己的代码全放到自己定义的唯一空间名字中，降低全局命名冲突的问题

  • 建议使用 let 关键字定义局部作用域

  • 在ES6引入了常量关键字（只读变量） const
